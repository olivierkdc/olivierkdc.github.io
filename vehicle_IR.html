Project Introduction

  Computer Vision is a currently one of the hottest fields in Machine Learning. In this personal project, I build a Convolutional Neural Network in order to classify whether an image depicts a vehicle. The resulting model built in this project attains an accuracy of 99%.


Data Exploration and Preprocessing

  The Data was obtained from Kaggle.com. We first performed some Preprocessing on the data to ensure proper image dimensions to be fed into our Machine Learning Model. As we are running a Convolutional Neural Network, it is crucial that the images all have the same dimensions. We therefore reshape the images to be have shape (64,64,3) and normalie the values to be between 0 and 1.
Next, we explore our Dataset in order to make sure that there are no blatant class imbalances.


Class Balance

  After preprocessing and randomly shuffling our data, we perform a small sanity check to verify that our data seems to be correctly labeled. In this case, a 1 indicates that the image depicts a vehicle while a 0 indicates that the image does not contain a vehicle.


Random Sample of Images in our Data

Machine Learning Modeling

  Now that the data has been preprocessed and verified, we are ready to begin modeling. In this case, we are utilizing Tensorflow and Keras workflows to build our Convolutional Neural Network. After experimenting, I found that the model did not need to be particularly complicated in order to achieve a high accuracy as a result. Therefore, the model contains two convolutional layers, and 2 Dense Layers.
  In the hope to avoid overfitting, I have included Callbacks to end the training once the model accuracy stops increasing for multiple epochs in a row. The model ended training after 22 epochs and restored the weights of the optimal epoch: 19.


Model Accuracy in Loss throughout Training Epochs

Conclusion

  The CNN model performed incredibly well with the data. While this project was interesting, I believe it to be quite simple. In the future, I would like to tweak this model in order to classify different types of vehicles as opposed to a binary category classification.
